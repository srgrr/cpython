# CPython With Built-in Garzon Paradox Solver
This Python implementation solves the Garzon paradox in bytecode generation time.
![garzon_mas_tonto](garzon_mas_tonto.jpg)

## Sample snippet

```python
for _ in range(10):
  print(garzon_mas_tonto)
```

Generates the following output

```
Alberto
Eduardo
Alberto
Eduardo
...
```

Note that you do not even need to define who is `garzon_mas_tonto`, the language figures it out by itself!
